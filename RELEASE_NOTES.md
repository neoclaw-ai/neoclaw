# Release Notes — v0.2.0

## What's New

- **Fact-based memory system** — This release is all about the memory system.
  There's a brand new structured memory system. NeoClaw remembers facts about
  you and recalls them at the relevant time.

- **`claw version` command** — Run `claw version` to print the current build version and commit hash.

## Bug Fixes
- Stability improvements
- Better debug logging
- Clearer system prompts
